
#Redis based minimal Database Management System

Implementation of a simple relational database management system,
using Redis as a key value store.

### Currently supported queries
> CREATE/DROP DATABASE

> CREATE/DROP TABLE

> USE DATABASE

> INSERT INTO TABLE

> DELETE FROM TABLE

### Working on

> CREATE INDEX FILE

### Queries to come
> SELECT +  JOIN

> SELECT +  GROUP BY
